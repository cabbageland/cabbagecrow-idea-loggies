# CabbageCrow Apple Shortcuts

CabbageCrow registers the `cabbagecrow://` URL scheme in the packaged macOS app.
In Apple Shortcuts, add an "Open URLs" action and use one of these URLs:

```text
cabbagecrow://mode/idle
cabbagecrow://mode/waving
cabbagecrow://mode/jumping
cabbagecrow://mode/waiting
cabbagecrow://mode/running
cabbagecrow://mode/review
cabbagecrow://mode/failed
cabbagecrow://size/up
cabbagecrow://size/down
cabbagecrow://size/reset
cabbagecrow://size/set?scale=1.35
cabbagecrow://dashboard/open
cabbagecrow://free-range/on
cabbagecrow://free-range/off
cabbagecrow://free-range/toggle
cabbagecrow://spark/pull
cabbagecrow://spark/add?text=Sketch%20tiny%20ritual%20cards
```

The app must be installed/opened once so macOS registers the URL scheme.
